# ESLint Config by Ponominalu.ru

Install ESLint and config:

```bash
npm i -D eslint \ 
eslint-config-standard \
eslint-plugin-import  \
eslint-plugin-node \ 
eslint-plugin-promise 
eslint-plugin-vue \ 
eslint-config-ponominalu
```

`.eslintrc` have to includes:

```json
{
  "extends": "ponominalu"
}
```

That's it!
